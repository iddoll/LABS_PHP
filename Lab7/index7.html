Дата виконання: 19.06.2025

Завдання

Завдання 1

Умова: Розробіть проєкт веб ресурсу відповідно до свого варіанту та розмістіть його на публіч ному ресурсі в мережі Інтернет. У звіті вкажіть URL адресу його розміщення. Варіант 2. Система управління студентськими записами 1. Створити базу даних MySQL для зберігання даних про студентів, курси та оцінки. 2. Реалізувати панель адміністратора для введення даних про студентів та їх успішність. 3. Вивести звіт про середній бал студентів по кожному курсу.
